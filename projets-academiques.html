Projets Académiques

Voici quelques projets académiques que j'ai réalisés au cours de ma formation.

Ces projets couvrent divers domaines tels que l'analyse de données, le développement web et la gestion de bases de données.

Conception d’une solution applicative en micro-services – 2024–2025

  • Analyse des besoins métier pour la refonte d’une architecture SI dans un contexte e-commerce.
  • Développement d’APIs REST avec FastAPI en architecture micro-services, chaque service ayant sa propre base.
  • Conteneurisation des services avec Docker pour faciliter le déploiement.
  • Intégration d’un message broker pour la synchronisation inter-services.
  • Mise en place de pipelines CI/CD avec GitHub Actions, stratégie de versioning avec GitFlow.
  • Tests automatisés (unitaires et intégration) avec couverture supérieure à 95%.

Technologies : FastAPI, Docker, MongoDB, GitHub Actions, GitFlow, Message Broker

Dashboard Power BI – Analyse de la consommation d’énergie en France (2025)

  • Exploration de données ouvertes issues d'Enedis (consommation d’électricité et de gaz par commune et par secteur d’activité).
  • Nettoyage et transformation des données (gestion des types, suppression des doublons).
  • Modélisation d’un tableau de bord interactif avec Power BI (KPI, slicers, cartes, bar charts, heatmaps).
  • Filtrage dynamique par année, région, secteur d’activité et filière (électricité/gaz).
  • Mise en forme professionnelle (thème de couleurs cohérent,indicateurs visuels...).

Technologies : Power BI, DAX, open data (data.gouv.fr), CSV, visual design

Acquisition et Traitement des Informations sur les Plus Grandes Banques du Monde - 2025

  • Extraction des données financières en utilisant Web Scraping avec requests et BeautifulSoup.
  • Transformation et conversion des données avec Pandas et numpy en utilisant des taux de change
  • Chargement des données dans un fichier CSV et une base de données SQLite3.
  • Exécution de requêtes SQL pour analyser les données stockées
  • Mise en place d’un système de journalisation avec datetime pour suivre l’exécution du script.
  • Projet réalisé dans le cadre d’un laboratoire pratique sur Coursera

Technologies: Pandas,Numpy, SQLite3, BeautifulSoup
